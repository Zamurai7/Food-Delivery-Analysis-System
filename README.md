# Food Delivery K-means & Association Rules Analysis 🍕

A comprehensive interactive Shiny web application for analyzing food delivery performance data using advanced machine learning algorithms including K-means clustering and Association Rules mining.

## Overview

This application provides data scientists and logistics analysts with powerful tools to:
- **Identify delivery patterns** through K-means clustering
- **Discover late delivery risk factors** using Association Rules
- **Visualize complex relationships** in delivery data
- **Make data-driven decisions** to improve delivery performance

Built with R Shiny for an intuitive, browser-based interface with no coding required.

## Features

### Machine Learning Algorithms
- **K-means Clustering** - Segment deliveries into homogeneous groups based on multiple features
- **Association Rules Mining** - Discover rules and patterns that lead to late deliveries
- **Pattern Discovery** - Understand which factor combinations increase delivery lateness risk

### Comprehensive Visualizations
- **8 Static Charts** - Distribution analysis, correlations, and comparisons
- **3 Dynamic Interactive Charts** - Real-time echarts4r visualizations
- **3D Analysis** - Multi-dimensional scatter plots using Plotly
- **Cluster Visualization** - PCA-based cluster representations

### Data Management
- **CSV Upload** - Simple file import interface
- **Data Preview** - Interactive table with sorting and filtering
- **Export Results** - Download clustered data and discovered rules
- **Error Handling** - Graceful handling of corrupted or invalid files

### User Experience
- **Modern UI Design** - Bootstrap 5 with Morph theme
- **Responsive Layout** - Works on desktop and tablet
- **Real-time Processing** - Instant algorithm execution
- **Intuitive Controls** - Sliders and buttons for parameter tuning
